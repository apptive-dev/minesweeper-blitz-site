# 💣 Minesweeper Blitz

A fast-paced modern remake of the classic Minesweeper puzzle game built for mobile devices.

Minesweeper Blitz combines the timeless strategic gameplay of Minesweeper with modern UI design, smooth performance, and multiple game modes.

Built to run on both iOS and Android.

---

## 📱 Features

• Classic Minesweeper gameplay  
• Multiple game modes  
• Smooth mobile-first UI  
• Touch-optimized controls  
• Statistics tracking  
• Optional premium upgrades  
• Ad-supported free version  

Game modes include:

### Classic Mode
Traditional Minesweeper gameplay where players clear the board without detonating bombs.

### Levels Mode
Progressive stages that increase in difficulty as the player advances.

### Challenge Mode
Timed or special-rule challenges designed for experienced players.

---

## 🧠 Gameplay Overview

Players uncover tiles on a grid while avoiding hidden mines.

Numbers on tiles indicate how many mines are adjacent.

Using logic and deduction, players must safely clear the entire board.

Detonate a mine and the game ends.
